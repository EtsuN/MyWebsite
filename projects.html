Maya Archaeology Project ROS C++


Project Overview

This project aims to support the archaeologists who have been studying the buried Maya temples for many years through excavating tunnels around the temples. Specifically, our team has worked on creating a system to generate 3D models of the tunnels that can accurately document the temple structures within a reasonable amount of time. Originally, the team was developing a system using single camera to record the tunnel excavation. However, there are problems in accuracy of the generated 3D model due to the system’s likeliness to lose track of itself while recording. We propose a dual-camera system that can solve this problem by increasing the accuracy of the generated 3D model through its wider field of view. Furthermore, the two cameras will reduce the problem of losing track.

This project is developed by a team of four students with a mentor.

Report PDF YouTube Video Source Code

October 2017 - December 2018

  • Attempted to use MCPTAM algorithm to develop dual-camera system using to record and produce point clouds, and discovered that MCPTAM is unpractical for actual usage.

    • SLAM algorithm

  • Tested on the accuracy of the 3D model.
